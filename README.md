# Otamesi
k8s+Vue+flask+psqlでつくる学習用コンテナ自動デプロイwebアプリケーション

Voyage group #サマーハッカソン制作物

*開発環境デプロイ方法*

1.リポジトリをダウンロード

2.`k8s`ディレクトリのマニフェストのmountパスを書き換える

3.kubectl apply -f ./k8s
